# ONEM — Plateforme de déclaration mensuelle

## Aperçu rapide

Vous pouvez prévisualiser la page de deux façons :

### Option 1 — Ouvrir directement le fichier

- Ouvrez `index.html` dans votre navigateur (double‑clic ou glisser‑déposer).

### Option 2 — Lancer un petit serveur local

```bash
python -m http.server 8000
```

Ensuite, ouvrez `http://localhost:8000/index.html` dans votre navigateur.

## Contenu

- `index.html` : la structure de la page.
- `style.css` : le design (typographie, grilles, cartes, responsive).
